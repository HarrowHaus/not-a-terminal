# Not A Terminal

A search engine for web templates with a natural language interface.
The intelligence is in the curation, not the computation.

## Architecture
User input → 23MB embedding model → cosine search → AST transform → live preview.
No generative AI. $0 runtime cost. Works on any device.

## Build
1. Open Claude Code, navigate to this folder
2. Paste Phase 0 from PROMPTS.md (creates the repo)
3. Continue through Phase 14

## Files
- CLAUDE.md — Claude Code master instructions
- BUILD_DOSSIER.md — Complete locked specification
- PROMPTS.md — Phase-by-phase build prompts (15 phases)
- AGENTS.md — Haiku sub-agent for git operations
- DESIGN_REFERENCE.html — Locked visual identity (open in browser)

## The Thesis
4,500+ professionally designed components. 600-900 unique section variants.
Infinite combinations. Zero AI generation.
Every piece was made by a human. Every combination is made for you.
